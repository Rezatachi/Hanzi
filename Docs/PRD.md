# Mandarin Drift PRD

## Product Vision
Mandarin Drift is an iPhone-first Mandarin learning app built around passive exposure, tiny study loops, and calm repetition. The product focuses on recognition, pronunciation, usage context, and daily recall without pressure-heavy gamification.

## Core Value
- One featured daily word or character.
- Local-first spaced repetition.
- Passive learning through widgets and optional wallpaper study mode.
- Original seed dictionary with learner-safe examples and memory hooks.
- No account required for core use.

## Core Surfaces
- Today
- Study
- Search
- Progress
- Profile / Settings
- Widgets
- Paywall
- Wallpaper generator

## Freemium Model
- Free: daily card, basic search, limited reviews, one widget mode, basic progress, saved limit.
- Premium: unlimited reviews, all widgets, wallpaper generator, advanced stats, unlimited saved cards, all categories, full dictionary expansion readiness.

## Privacy
- Local-first by default.
- Optional backend adapter only.
- No third-party analytics SDK by default.
- Export/delete architecture accounted for in settings and repo design.
